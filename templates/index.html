The goal of the Air Fare Prediction project is to develop a model that can accurately predict the prices of airline tickets. This project utilizes data analysis, machine learning, and statistical modeling techniques to analyze historical flight data and make predictions on future flight prices.


The Air Fare Prediction project is aimed at helping travelers plan their trips more effectively by providing insights into the potential price trends of flight tickets. This model predicts the price of the flight based on some parameters like total stops, journey day, journey month, airline, source, destination, etc. We have trained this model using the random forest regressor and after training, fine-tuned the model through hyperparameter tuning. Then, the model is saved and deployed using a Flask application on localhost.

Frequently Asked Questions (FAQs)

What is the purpose of this project?
The goal of this project is to predict airline ticket prices using machine learning models, helping travelers plan their trips efficiently.
Which models are used in this project?
The models used include ExtraTree Regressor, Linear Regression, XGBoost, and Random Forest Regressor.
How accurate is the best model?
The Random Forest Regressor achieves the highest accuracy of 93.80% with an R² score of 0.90.
How is this model deployed?
The model is deployed using Flask on a local server to provide real-time fare predictions.
